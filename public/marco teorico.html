Marco Teórico de HTML

¿Qué es HTML?

Imagina que estás construyendo una casa. Necesitas un esqueleto, unos cimientos, paredes, un techo... Esa estructura básica es el HTML de una página web.

  • HTML son las siglas de HyperText Markup Language (Lenguaje de Marcado de Hipertexto).
  • No es un lenguaje de programación. No puedes hacer cálculos matemáticos ni lógica compleja con él.
  • Es un lenguaje de "marcado". Su único trabajo es describir y dar estructura al contenido de una página web. Le dice al navegador: "esto es un título", "esto es un párrafo", "esto es una imagen", etc.

¿Cómo funciona? Las Etiquetas (Tags)

HTML funciona a través de "etiquetas" (en inglés, tags). Las etiquetas son como comandos que le das al navegador, y se escriben entre corchetes angulares (< y >).

La mayoría de las etiquetas vienen en pares:

  1. Una etiqueta de apertura: <p> (le dice al navegador: "aquí empieza un párrafo").
  2. Una etiqueta de cierre: </p> (le dice al navegador: "aquí termina el párrafo").

El contenido va en medio de las dos: <p>Este es el texto de mi párrafo.</p>

La Estructura Básica de TODA página HTML

Cualquier archivo HTML profesional tiene esta estructura mínima. Es como el ADN de una página web:


<!DOCTYPE html>
<html>
  <head>
    <!-- Aquí va información para el navegador (no se ve en la página) -->
  </head>
  <body>
    <!-- Aquí va TODO el contenido que el usuario SÍ ve -->
  </body>
</html>
  • <!DOCTYPE html>: Es la primera línea. Le dice al navegador "Oye, este es un documento HTML5 moderno".
  • <html>: Es la etiqueta raíz. Todo lo demás vive dentro de ella.
  • <head> (Cabeza): Contiene información "invisible" para el usuario, como el título que aparece en la pestaña del navegador.
  • <body> (Cuerpo): Es la parte más importante. Todo lo que quieras que el usuario vea (texto, imágenes, botones, formularios) debe ir aquí dentro.
